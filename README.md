⸻

1. Proje Açıklaması

PharmaGo, kullanıcıların ilaç ve sağlık ürünlerine hızlı, güvenli ve kolay bir şekilde ulaşmasını amaçlayan mobil tabanlı bir sağlık teslimat uygulamasıdır. Sistem sayesinde kullanıcılar reçeteli veya reçetesiz ürünlerini uygulama üzerinden sipariş edebilmekte, e-Nabız entegrasyonu ile reçetelerine erişebilmekte ve siparişlerini kapılarına kadar teslim alabilmektedir.

Proje özellikle yaşlı bireyler, kronik hastalar, yoğun çalışan kişiler ve evden çıkmakta zorlanan kullanıcılar için geliştirilmiştir. Amaç sağlık ürünlerine erişimde yaşanan zaman kaybını ve fiziksel zorlukları ortadan kaldırmaktır.

7. Kurulum Adımları

Flutter Kurulumu

1. Flutter SDK kurulumu yapılır.
2. Android Studio yüklenir.
3. Flutter ortam değişkenleri tanımlanır.
4. Proje klasörü açılır.
5. Terminal üzerinden aşağıdaki komut çalıştırılır:

flutter pub get

6. Uygulama çalıştırılır:

flutter run

Backend Kurulumu

1. Node.js yüklenir.
2. Backend klasörüne girilir.

cd backend

3. Gerekli paketler yüklenir.

npm install

4. Sunucu başlatılır.

npm start

11. Sonuç

PharmaGo, sağlık sektöründe dijital dönüşümü destekleyen yenilikçi bir mobil uygulama projesidir. Kullanıcıların ilaç ve sağlık ürünlerine güvenli, hızlı ve kolay şekilde ulaşmasını sağlayarak sağlık hizmetlerinin erişilebilirliğini artırmayı hedeflemektedir. e-Nabız entegrasyonu, canlı sipariş takibi ve hızlı teslimat özellikleri sayesinde kullanıcı deneyimini iyileştiren modern bir çözüm sunmaktadır.
